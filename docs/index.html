Informatik-Zitate

Diese Webseite ist eine kleine Sammlung von Zitaten/Sprüchen aus dem Bereich der Informatik.

Sie wurde mit jQuery Mobile erstellt. Diese und die Seiten mit den Zitaten sind in einer einzigen HTML-Datei enthalten und durch DIV-Elemente ausgezeichnet. Dadurch muss der Browser nur eine HTML-Datei laden, um alle Seiten auf das Gerät zu bringen. jQuery Mobile basiert auf der JavaScript-Bibliothek jQuery.

Über diese HTML-App

 

Die für die Nutzung von jQuery Mobile notwendigen JavaScript-Bibliotheken und die zugehörige CSS-Datei (Stylesheet) werden von einem Content Distribution Network (CDN) nachgeladen. Diese Dateien können aber auch auf dem Web-Server mit der Web-App abgelegt werden.

Alle angezeigten Seiten sind in einem HTML-Dokument enthalten, wo sie durch <div>-Elemente mit entsprechenden Attributen ausgezeichnet sind. Hierzu definiert jQuery Mobile sog. "Custom Data Attributes", die mit "data-" beginnen müssen, z.B. "data-role" oder "data-theme".

Durch Betätigung des "i"-Buttons rechts oben auf dieser Seite wird ein Dialog mit Versions-Informationen geöffnet.
